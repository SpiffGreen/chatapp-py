## ChatApp

A simple chat application written in python

### Technologies
- Python
- Socketio
- Tailwind
- MySQL

### Features:
- [x] Chat box
- [x] Find friends
- [] Message status
- [x] Add friend
- [x] Remove friend
- [] Delete Message
- [x] Profile
- [x] Friend list
- [x] Login/logout

### Python virtualenv
create and activate a python virtual environment
```sh
$ python3 -m venv venv
$ . venv/bin/activate
```

To deactive - simply run the code below
```sh
$ deactivate
```

### Dependencies Management
Install Dependencies
```sh
$ python -m pip install -r requirements.txt
```
Save Dependencies
```sh
$ python -m pip freeze > requirements.txt
```

### Run Project
```sh
$ python ./src/main.py
```
